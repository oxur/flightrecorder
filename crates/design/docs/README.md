# FlightRecorder Design Documentation

This directory contains all design documentation for the FlightRecorder project.

## Document Organization

Documents are organized using `oxur-odm` (Oxur Design Document Manager) by status:

- `00-inbox/` - New ideas and proposals
- `01-draft/` - Documents under active development
- `02-review/` - Documents ready for review
- `03-approved/` - Approved but not yet implemented
- `04-rejected/` - Rejected proposals
- `05-active/` - Currently implemented and maintained
- `06-superseded/` - Replaced by newer documents
- `07-deprecated/` - No longer relevant
